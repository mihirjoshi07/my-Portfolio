Mihir's Blog

Mihir's blog is an online platform that allows users to create accounts and log in to the application. Once logged in, users have the ability to post their own content. This includes the ability to create new blog posts, edit their own posts (subject to proper authorization), and view posts authored by other users.

Welcome to the main page of Mihir's Blog, where you'll find an existing post titled "iPhone '15," authored by a user named "test." The homepage is a hub for a diverse array of posts contributed by various users.

Here, you can explore a chronological list of posts, each clearly attributed to its respective author, providing insight into the wealth of perspectives within the Mihir's Blog community.

In addition to the posts, you'll find two essential options on the homepage:

Register: If you're new to Mihir's Blog and eager to become a member, simply click on the "Register" button. This will initiate the registration process, allowing you to create a personalized account with a unique username and password.
Login: For those who have already registered, the "Login" button grants access to the platform using your registered credentials. Logging in empowers you to contribute your own posts, modify them (with appropriate authorization), and engage with the content posted by other users.

Let's initiate the user registration process by creating a new user account. We've defined the username and password for this user as "admin." Simply click the "Register" button to save this user's information into our MongoDB database.

Let's take a look at the backend infrastructure, which relies on MongoDB Atlas, a cloud-based web application for MongoDB. Additionally, for security purposes, we've implemented the bcrypt library to encrypt user passwords. This library is capable of hashing and salting passwords to generate a randomized, secure, and encrypted password for each user.

After successfully registering a user, let's proceed to log in using the same username and password in order to access the blog page.

Once you've successfully logged in, you'll notice two options at the top: "Create Post" and "Logout." With these options available, you can now create new posts and edit existing ones, provided you have the proper authorization. This grants you the ability to actively contribute to the blog and manage your content as you see fit.

"Having created a post as the admin user, it's now possible to edit posts that belong to a specific user. For instance, if we log in as another user named 'test,' we can then update the post about the Apple iPhone."

Great, you've successfully edited the title of the post to "APPLE." To continue, let's log out from the "test" user account and view the dashboard or homepage. Logging out will allow us to see the changes made to the post title and navigate through the application without an active user session.
