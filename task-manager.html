Task Manager

A task manager is an application used to document everyday tasks. It allows for the modification of task names and the deletion of tasks as needed. You also have the ability to mark tasks that you've already completed as done.

Tech Stack : MERN(MongoDb, Express, Node)

This is the homepage of a task management application hosted on Render deployment. On this homepage, users can utilize an input card to enter their tasks into a designated input field. Once they've entered a task, they can click the submit button to save it to the backend storage.

Upon task submission, the task is stored in a MongoDB database. Users can then retrieve their tasks by fetching data from the backend using an API created with Express.js (Node.js) and displaying it on the frontend.
